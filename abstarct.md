# ABSTRACT

This project presents the design and development of PayGuard, a mobile money fraud detection and monitoring system. The study was motivated by the challenges faced in manually reviewing mobile money transactions, where suspicious activity can be missed due to high transaction volumes, delayed checking, scattered records, and limited reporting tools. PayGuard was developed to support finance users by analysing transaction records, assigning fraud risk scores, generating alerts, and presenting key information through a simple dashboard.

The system was implemented as a web-based prototype using a React and TypeScript frontend, a Python Flask backend, and an SQLite database. Synthetic mobile money transaction data was used for development and testing because real financial data is sensitive and difficult to access for academic work. A machine learning model was trained using transaction features such as amount, transaction type, account behaviour, device information, location, and transaction history. The trained model produces a risk score that helps identify transactions that may require further review.

PayGuard provides users with dashboard summaries, transaction records, alert monitoring, account profile information, and model performance metrics. The system does not replace human judgement, but it assists users by highlighting suspicious transactions and making fraud review faster and more organised. Testing showed that the prototype met its main functional requirements, including transaction scoring, alert creation, data storage, dashboard display, and account review.

The project demonstrates how machine learning, database storage, backend processing, and dashboard visualisation can be combined to improve mobile money fraud monitoring. Although the prototype uses synthetic data and does not connect to live mobile money providers, it provides a useful foundation for future development using real approved transaction data, stronger security controls, user authentication, and integration with financial platforms.
